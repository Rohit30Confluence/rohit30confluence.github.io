## 🔷 About

Portfolio for **Rohit Dinde** — *dr.confluence* — Engineer & Open Source Contributor, built around a systems-first, security-and-automation narrative: from operations to founding teams, from scripting to AI, from copy-pasted projects to building everything from scratch.

**Stack:** HTML5 · CSS3 · Vanilla JS (custom cursor, animated node canvas, scroll reveals, count-up stats, filterable project grid)
**Hosted:** GitHub Pages (auto-deploys on every push to `main`)
